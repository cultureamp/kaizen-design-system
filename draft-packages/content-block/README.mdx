## To keep in mind

* `ContentBlock` will match the width and alignment of the `TitleBlockZen`
  content
* `ContentBlock` should be used as a wrapper for most page content below the
  title block
* `ContentBlock` should **always** be wrapped in a `ContentBlockContainer` to
  ensure it is spaced correctly
* `ContentBlockContainer` will stretch to the edge of the viewport, and so it
  can be used as a container for full-width elements (backgrounds)
