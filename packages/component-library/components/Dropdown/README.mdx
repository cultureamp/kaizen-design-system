## To keep in mind

*   Dropdowns contain links or actions, not selected values.
*   [Focus indicator](/guidelines/interaction-states)
  *    On keyboard navigation show focus indicator on top level selector.
  *    For menu select options show focus indicator on keyboard navigation and on click (not hover).


## When to use and when not to use

<WhenToUseAndWhenNotToUse>
<WhenToUse>

*   When you’re short on space and need to provide further links or actions.

</WhenToUse>
<WhenNotToUse>

*   When you’re actually selecting a value from a list of options, use a [Select](/components/select) instead.

</WhenNotToUse>
</WhenToUseAndWhenNotToUse>
