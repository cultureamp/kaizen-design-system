## Overview

Pagination separates large bodies of content into separate pages. You can access each page via a shared index of links.

### Pagination and Directional Link `disabled`

A disabled link prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may not read aloud by a screenreader.
