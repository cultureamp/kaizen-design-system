## Bolding

When you need to bold text, first ask if you are bolding to add emphasis or for stylistic reasons.

If you are adding emphasis to the text, use the `<strong>` tag. E.g. `<Paragraph>The quick brown fox jumps over the <strong>lazy</strong> dog.</Paragraph>`

If you are just bolding for style, use Tailwind with `classNameOverride` instead. E.g. `<Paragraph variant="body" classNameOverride="font-weight-paragraph-bold">The quick brown fox jumps over the lazy dog.</Paragraph>`
