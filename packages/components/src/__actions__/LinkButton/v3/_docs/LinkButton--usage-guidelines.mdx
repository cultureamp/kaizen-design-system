## Overview

`LinkButton` allows users to navigate to another page or resource within a web page or application. It shares the same visual styles and interaction states as the <LinkTo pageId="actions-button-button-v3-usage-guidelines--docs">Button</LinkTo> component, but is intended for navigation purposes or downloading documents.
