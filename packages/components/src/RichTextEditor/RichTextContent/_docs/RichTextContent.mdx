## Usage

A common use case of `RichTextContent` is to display user generated output in a read-only state.

<Canvas of={RichTextContentStories.ReadOnly} />

This should not be used toggling between active and inactive states, such as editing and saving content. Instead we recommend using the [EditableRichTextContent](/docs/components-richtexteditor-editablerichtextcontent--docs) pattern, which more clearly indicates interactivity to the user.
