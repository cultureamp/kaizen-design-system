## Overview

A simple wrapper to wrap form field components and their related siblings. This is a primitive used in components like `TextField`, `TextArea`, `Slider`, etc.

## API

### Inline
Changes the FieldGroup to an inline element with no bottom margin. This can be seen in the `Slider` [component](/?path=/docs/components-slider--docs).
