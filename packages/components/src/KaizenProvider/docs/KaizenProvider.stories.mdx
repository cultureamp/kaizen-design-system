# KaizenProvider

Provides the necessary theming context for all components, as well as facilitate the use of global UI components and translations.

## Usage

Ideally, the `KaizenProvider` should wrap your entire application, and that there only be one per application.

```jsx
import { KaizenProvider } from '@kaizen/components';

<KaizenProvider>
  <App />
</KaizenProvider>
```
