# Managing the theme

Out of the box, the `KaizenProvider` utilises the `defaultTheme` from `@kaizen/design-tokens`. However if you want to roll your own theme, we provide a `ThemeManager` function which can apply your custom theme.

```jsx
export const customTheme = {
  themeKey: "custom",
  spacing: {
    md: "1rem",
  },
};

<KaizenProvider theme={customTheme}>
  <App />
</KaizenProvider>
```

## How to switch themes

You can use any method you want to switch themes, one way is React state.

Example:

```js
import { defaultTheme } from "@kaizen/components"
const [useCustomTheme, setUseCustomTheme] = useState(false)

useEffect(() => {
  // Conditional logic to change the boolean
  setUseCustomTheme(data ? true : false)
}, []);

<KaizenProvider theme={useCustomTheme ? customTheme : defaultTheme}>
  <App />
</KaizenProvider>
```
