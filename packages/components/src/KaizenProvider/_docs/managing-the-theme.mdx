# Managing the theme

Out of the box, the `KaizenProvider` utilises the `defaultTheme` from `@kaizen/design-tokens`. However if you want to roll your own theme, we provide a `ThemeManager` function which can apply your custom theme.

```jsx
import { ThemeManager } from '@kaizen/components'

export const themeManager = new ThemeManager({
  themeKey: "custom",
  spacing: {
    md: "1rem",
  },
});

<KaizenProvider themeManager={themeManager}>
  <App />
</KaizenProvider>
```

## How to switch themes

You can use the `setAndApplyTheme` method to change the current theme to either Heart or a customised object.

Example:

```js
themeManager.setAndApplyTheme(heartTheme)

// or make your own modifications
themeManager.setAndApplyTheme({
  ...heartTheme,
  themeKey: "custom",
  spacing: {
    ...heartTheme.spacing,
    md: "1rem",
  },
})
```
