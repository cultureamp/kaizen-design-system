# Documentation Generation Context

## Package Overview

@kaizen/components is Culture Amp's Kaizen component library - a comprehensive collection of React UI components that power Culture Amp's products. It provides accessible, well-designed components following the Kaizen Design System guidelines.

## Target Audience

Frontend developers building Culture Amp products who need to implement consistent, accessible UI components. Users are expected to be familiar with React and TypeScript.

## Documentation Style

- Include practical usage examples with TypeScript
- Show all available props with their types and default values
- Explain accessibility considerations for each component
- Include examples of common patterns and compositions
- Show integration with KaizenProvider for theming

## Key Patterns

- React functional components with hooks
- TypeScript for type safety
- CSS Modules for component styling
- Compound component patterns for complex components
- React Aria for accessibility primitives
- i18n support via locales

## Example Guidelines

- Show basic usage first, then advanced patterns
- Include accessibility-focused examples
- Demonstrate common prop combinations
- Show composition with other Kaizen components
- Include examples with and without KaizenProvider context
