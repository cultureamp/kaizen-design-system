# How to use design tokens in React components


## 1. Setup a KaizenProvider

Ensure you have followed the steps in [KaizenProvider](/story/components-kaizenprovider--page) to setup your application for using themes.

## 2. Start using design tokens with the `useTheme` hook

Once the theme context is available, the `useTheme` hook will allow you to subscribe components and give access to the current theme, and re-render whenever it changes.

Example:

```jsx
import { useTheme } from "@kaizen/design-tokens"

const Component = () => {
  const theme = useTheme();

  return (
    <div style={{
      backgroundColor: theme.color.purple["500"] // #844587
    }}>
      Theme'd component
    </div>
  )
}
```


## Customising the theme

Out of the box, the `KaizenProvider` utilises the `defaultTheme` from `@kaizen/design-tokens`. However if you want to roll your own theme, we provide a `ThemeManager` function which can apply your custom theme.

```jsx
import { ThemeManager } from '@kaizen/components'

export const themeManager = new ThemeManager({
  themeKey: "custom",
  spacing: {
    md: "1rem",
  },
});

<KaizenProvider themeManager={themeManager}>
  <App />
</KaizenProvider>
```
