# Access tokens via JSX

## 1. Set up a KaizenProvider

Ensure you have followed the steps in <LinkTo pageId="components-kaizen-provider-installation">KaizenProvider</LinkTo> to set up your application for using themes.

## 2. Use CSS Vars

Because we use CSS vars, you can utilise them in your JSX as you would in CSS.

For example:

```jsx

const Component = () => (
  <div style={{
    backgroundColor: vars(--color-purple-500) // #844587
  }}>
    Custom Kaizen theme component
  </div>
)
```
