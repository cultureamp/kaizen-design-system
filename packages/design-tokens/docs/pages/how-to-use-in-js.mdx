# Access tokens via JSX

## 1. Set up a KaizenProvider

Ensure you have followed the steps in <LinkTo pageId="components-kaizen-provider-installation">KaizenProvider</LinkTo> to set up your application for using design tokens.

## 2. Use Tokens

### In JSX via CSS Vars

Because we use CSS vars, you can utilise them in your JSX as you would in CSS.

For example:

<Source
  language="tsx"
  code={`
const Component = () => (
  <div style={{
    backgroundColor: vars(--color-purple-500) // #844587
  }}>
    Custom Kaizen theme component
  </div>
)
  `}
/>

### In JS functions

You can access the tokens directly:

<Source
  language="tsx"
  code={`
import { tokens } from "@kaizen/design-tokens/js"

const myAmazingFunction = () => {
  const mainColor = data ? tokens.colors.purple[500]: tokens.colors.purple[300];

  return mainColor;
}
  `}
/>
