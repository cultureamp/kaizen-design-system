# Documentation Generation Context

## Package Overview

@kaizen/tailwind provides Tailwind CSS presets that integrate Kaizen design tokens into Tailwind projects. It enables developers to use Kaizen's design system values (colors, spacing, typography) through Tailwind's utility class syntax.

## Target Audience

Frontend developers using Tailwind CSS who want to apply Kaizen Design System values consistently. Users should be familiar with Tailwind CSS configuration and utility classes.

## Documentation Style

- Show how to configure Tailwind with Kaizen presets
- Provide examples of generated utility classes
- Explain the mapping between Kaizen tokens and Tailwind utilities
- Include integration examples with @kaizen/components

## Key Patterns

- Tailwind preset configuration
- Custom spacing scale based on Kaizen tokens
- Custom color palette from Kaizen design tokens
- Height utilities for consistent component sizing

## Example Guidelines

- Show Tailwind configuration setup
- Demonstrate available utility classes
- Include examples of common patterns (spacing, colors, heights)
- Show integration with existing Tailwind projects
