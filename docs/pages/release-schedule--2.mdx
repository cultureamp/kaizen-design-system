# Release schedule

Upcoming releases for the Kaizen Design System.

## Glossary

- **Current**: The version of the component that is exported in the main package entrypoint (`@kaizen/components`).
- **RC (Release Candidate)**: The version of the component that will supersede the **Current** version.
- **Component migrations**: A list of components that will have their **RC** version replace the **Current** version in the release.
  - The **Current** version will be deprecated, then deleted in the next major release.
  - _(^ OR)_ The **Current** version will be deleted.
  - **Codemod**: A script that can be run to migrate component usages to from the **Current** API to the **RC** API.
