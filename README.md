## Scripts
Command | Summary
:- | :-
`yarn setup` | Install dependencies
`yarn reset` | Clean and reinstall dependencies
`yarn compile` | Run typechecks
`yarn lint` | Run linters
`yarn lint:fix` | Run linters, auto-fixing violations
`yarn test` | Run unit tests
